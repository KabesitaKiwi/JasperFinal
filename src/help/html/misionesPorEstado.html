4 - Misiones por estado

Genera un informe que lista misiones agrupadas o ordenadas por estado, incluyendo la nave asociada.

Qué hace

  • Ejecuta una consulta en Java para obtener: estado, descripción y nombre de nave.
  • Crea un JRBeanCollectionDataSource con objetos MisionEstadoInfo.
  • Rellena el JRXML con el datasource.
  • Abre visor y exporta a PDF.

JRXML

  • misionesPorEstado.jrxml

Datos / Campos

  • estado
  • descripcion
  • nombre_nave (si no tiene, aparece “Sin asignar”)

Parámetros

  • p_fecha: fecha actual
  • p_registro: código REG-...
