1 - Listado de naves

Genera un informe con el listado completo de naves de la base de datos.

Qué hace

  • Muestra las naves registradas (id, nombre, clase, capacidad, etc. según tu JRXML).
  • Abre el visor de Jasper y exporta el informe a PDF.

JRXML

  • listado_naves.jrxml

Parámetros

  • p_fecha: fecha actual
  • p_registro: código REG-...

Notas

No requiere rellenar campos.
