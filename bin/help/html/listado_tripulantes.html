2 - Listado de tripulantes

Genera un informe con el listado de tripulantes, incluyendo la nave asignada.

Qué hace

  • Ejecuta una consulta en Java (PreparedStatement) para obtener tripulantes + nave.
  • Crea un JRBeanCollectionDataSource con objetos TripulanteInfo.
  • Rellena el JRXML usando el datasource (sin conexión directa en Jasper).
  • Abre visor y exporta a PDF.

JRXML

  • listadoTripulantes.jrxml

Datos / Campos

  • nombre_tripulante
  • rango
  • nombre_nave (si no tiene, aparece “Sin asignar”)

Parámetros

  • p_fecha: fecha actual
  • p_registro: código REG-...
