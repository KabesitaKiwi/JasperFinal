5 - Misiones de una nave (por ID)

Genera un informe con las misiones filtradas por una nave concreta.

Qué hace

  • Lee el valor del campo de texto ID nave (a la derecha del botón).
  • Valida que sea un número.
  • Envía el parámetro p_id_nave al reporte.
  • Rellena el JRXML con la conexión JDBC.
  • Abre visor y exporta a PDF.

JRXML

  • misionesDeUnaNave.jrxml

Parámetro obligatorio

  • p_id_nave: ID de la nave (entero)

Parámetros adicionales

  • p_fecha: fecha actual
  • p_registro: código REG-...

Errores comunes

  • Si el campo está vacío: pide “Introduce el ID de la nave”.
  • Si no es número: “ID de nave no válido”.
